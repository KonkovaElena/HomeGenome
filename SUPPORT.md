# Support

## What This Repository Supports

HomeGenome supports:

- questions about repository structure and documentation;
- corrections to facts, links and citations;
- discussion of architecture, tooling and publication readiness.

## What This Repository Does Not Support

- medical advice;
- interpretation of your personal genome, BAM, VCF or pharmacogenomics profile;
- review of real patient data;
- emergency or clinically actionable decision support.

See also: [docs/reference/research-and-safety-boundary.md](docs/reference/research-and-safety-boundary.md).

## Where To Ask

- use Issues for documentation defects, factual corrections and focused feature proposals;
- use pull requests for concrete edits;
- use [SECURITY.md](SECURITY.md) for anything sensitive or publication-risk related.

## Before Opening An Issue

1. read [README.md](README.md);
2. check [docs/evidence/publication-readiness-audit-2026-04-21.md](docs/evidence/publication-readiness-audit-2026-04-21.md);
3. avoid attaching private biological data or large generated outputs.
